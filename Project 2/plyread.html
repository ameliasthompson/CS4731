Please use a browser that supports the "canvas" tag.
  • Load a file to display the model on the canvas.
  • Press x and c to toggle translating the model across the x axis.
  • Press y and u to toggle translating the model across the y axis.
  • Press z and a to toggle translating the model across the z axis.
  • Press r to toggle model rotation.
  • Press b to toggle model pulse mode.
